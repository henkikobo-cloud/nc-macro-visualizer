NC Macro Visualizer は、FANUC系NCマクロを静的解析し、 NCコードを「何をしている処理か」が分かる一般的なプログラムフローチャートへ翻訳する読解支援ツールです。

Problem

NCマクロは、現場の知識が詰まるほど読みにくくなる

01

読めない

変数番号、ラベル、条件分岐、GOTOが増えるほど、処理の流れを追うだけで時間がかかります。

02

属人化

作った本人の頭の中にだけ意味が残り、保守・改善・引き継ぎのたびに確認コストが膨らみます。

03

Mコード機械依存

Mコードは機械、PMC、ビルダー設定に依存します。未知のMコードを断定すると誤読につながります。

フローチャート

N番号ではなく処理内容へ翻訳して表示します。

値のつながり

変数がどの行で入り、どの行で使われるかを一覧で確認できます。

保存用メモ

確認内容をMarkdown形式で保存・共有できます。

Scope

このツールはNCエミュレータではありません

やること

NCマクロを静的解析し、初心者でも追える一般的なプログラムフローチャートへ翻訳します。

やらないこと

  • 実機動作保証
  • 加工シミュレーション
  • PMC再現
  • 工具軌跡、干渉、衝突、サイクルタイム推定
  • 機械固有Mコードの意味の推測

Feedback

このツール、使いますか？

製造現場でNCマクロを扱っている方の感想・要望を歓迎します。 未対応のNC方言、読み取りたい構造、出力形式への要望は GitHub Issue で共有してください。
